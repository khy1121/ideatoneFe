1️⃣ 가독이챗 메인 홈화면 (채팅목록)

<aside>

1. 하단 네비바에서 ‘가독이챗’을 클릭하면 뜨는 화면이다.
2. 오른쪽 상단에 새로운 노트 작성하기 버튼을 누르면 본격적인 가독이챗 화면이 생성된다. 
3. 가독이 채팅 기록이 최신 순부터 정렬된다.
4. **채팅 기록 내역을 클릭하면 해당 채팅 기록으로 넘어간다. - 가능할까요 ..? ?**
5. 맨 아래 채팅 기록까지 스크롤 형식으로 진행 (제한X)
</aside>

2️⃣ 가독이챗 입력 중 화면

<aside>

1. 가독이 이미지가 기존 자리에서 벗어나지 않은 채로 위아래 둥둥 떠다닌다.
2. 하단에 채팅창을 클릭하면 타자판이 올라오면서 네모박스 외곽선이 노란색으로 채워진다. 
3. 채팅을 입력하면 그에 맞게 채팅 박스도 위로 점점 늘어난다 
    1. 제한: 화면에 절반까지만 늘어나는 것이 가능하다
4. 채팅 보내기 : 
    1. 타자판이 올라와있는 채로도 보내기 버튼을 누르면 채팅이 보내진다.
    2. 보내기 버튼을 누르는 순간 타자판은 내려감
    3. 아무 화면을 눌러도 타자판은 내려감
5. 오른쪽에 업로드 화살표 버튼을 누르면 채팅이 보내진다. 
</aside>

3️⃣ 가독이챗 입력 완료 후 토론 화면

<aside>

1. 사용자가 보낸 채팅은 말풍선이 우측에 뜨고, 글자는 왼쪽 나열된다. 
2. AI가 대답을 생성하는 로딩 표시로 가독이 이미지가 역시나 기존 자리에서 둥둥 떠다니고, 겉 테두리에 로딩 원이 빙빙 돈다. 
3.   가독이가 분석을 끝내는대로 가독이가 보낸 채팅이 좌측에 뜬다.  
    1. 왼→ 오 방향으로 한글자씩 타이핑 되는 것처럼 표시 가능한가요 ?? 만약 구현하는 게 어렵다면 그냥 한번에 쑥 채팅 박스 안에 나오는 거로 해도 충분합니다 !! 
4. 사용자는 언제든 원할 때 채팅을 종료할 수 있다. 
    1. 따로 종료하는 버튼은 X
    2. 백버튼 누르면 다시 가독이챗 메인 화면으로 돌아간다. 
5. 왼쪽 상단에 계속 떠있는 가독이는 디자이너분이 추후 생각 예정 
</aside>
